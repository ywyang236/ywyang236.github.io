5-1. 閱讀心理相關書籍

5. 請描述一件讓你產生明顯負面情緒的事情，你如何處理該情緒？

我是個高度敏感的人，對情緒的反應也很敏銳，也曾經歷「情緒處理撞壁」很長一段時間，而「轉移注意力、接觸人群」對我來說很有效。為了能用不傷害他人的方式，面對他人的負面情緒帶給我的影響，我付費聽了蔡康永的201堂情商課（喜馬拉雅），又看了許多心理相關的書籍，像是《誰搬走了我的乳酪》、《情緒勒索》、《也許你該找人聊聊》、《情緒的解剖圖鑑》等，從中找方法，很積極的去認識情緒，以及找出合適的解決方法。

以我個人的例子來說，大四上學期，未來規劃產生劇烈變動，產生長達半年的消極情緒，當時的解決方法是轉移注意力，去接觸人群，不要讓自己對生活完全絕望，所以到學校的通識中心當工讀生、當助教，在大四畢業前重新找回對人生的希望。

第二個例子，是因為疫情導致公司虧損半年，資遣85%的員工後辦理歇業，很長一段時間沒有收入，產生巨大的自我懷疑，長達一年走不出負面情緒，甚至逃離都市住到鄉下田邊。後來，將注意力轉移到養水族寵物「米蝦」上，這是一種小型的彩色觀賞蝦，每天花 6～8 小時坐在水族箱前面盯著蝦看，並加入觀賞蝦社群分享相關知識，獲得米蝦養殖專家的肯定、水族社群的肯定，從中找到成就感，這才重新拿回人生的主導權，承認過去的失敗，冷靜地分辨可控與不可控的失敗成因。

6. 關於這份申請網頁，請分享一個你開發時的技術心得。

我想分享「相冊功能」的開發過程與心得。

一開始只是想展示幾張照片自動輪播，完成後看起來過於簡單，想讓使用者能左右滑動照片，所以又找了第二個版本，也成功設計出理想中的樣式，但又感覺不太滿足，希望能同時顯示文字，這才確定最後的版本。

因為是第一次把 Slideshow 塞進卡片裡，要兼顧內外版型，又要讓功能正常發揮，所以花很多時間調整。從發想到測試成功，換其他方法後再測試，一直到最後一版的完成，4～5 小時就這樣不見了，期間執著到忘記吃午餐，為了身體著想不得不趕快去吃晚餐，又趕回來完成後續。

每個「第一次」都特別花時間，但第二次開始就能在幾分鐘內快速完成想要的效果，一直摸索改動的過程是很令人開心的，雖然只是個小東西，但透過實作又學到了很多該注意的地方，像是：有記得到 cdn.js 找 w3-css 的 script 連結、把 script 放到正確的位置、盡量讓每張照片的副檔名一致比較好引用，以及使用這些模板工具的時候，一定要有耐心的閱讀網站的英文解釋，每一個字都有其存在的意義。

我在今年四月，完成人生第一篇技術分享的文章（在 Blogger 使用響應式表格的方法），接下來也會持續分享。

8. 其他想要對我們說的事情？

我預計在年底前完成一款名叫「觀察記錄簿」的 App，這是一款為科學 Maker 社團（全台灣最大的科學社團，目前有 8.3 萬人）的科學儀器而設計的 App。

規劃使用 Python 訓練 VGG16 深度學習模型辨識比例尺，在 React Native 框架串連此 api，讓手機鏡頭在手機顯微鏡的視野下，擁有自動辨識倍率的功能，並且可以幫照片增加自定義浮水印與 CC 授權浮水印功能。

目前剛確認計畫的可行性，以及簡單畫出 App 的 icon 樣式，也透過 YouTube 影片了解訓練的操作過程，以及透過 Udemy 課程了解 React 框架，下個階段會開始訓練此 App 的核心技能「辨識比例尺」，開始拍一萬張照片，一層一層的建立 VGG16 模型。

回顧過往的經驗，我是屬於實作能力優於理論能力的人，執行能力很強，再加上這段時間密集接觸程式語言，相當確定成為軟體工程師確實能發揮此優勢。另一個優勢，在於我是一個很願意為團體、為社團貢獻的人，很期待能加入 WeHelp，和一群有共同目標的人一起交流，為團體貢獻一份心力。
